# Sistema-de-OS-de-Oficina-Mec-nica
Modelagem de banco de dados para sistema de Ordem de Serviço de oficina mecânica, envolvendo clientes, veículos, serviços e mecânicos. Foram aplicados relacionamentos 1:N e N:N com entidades associativas, garantindo normalização, integridade referencial e uso adequado de tipos de dados para valores monetários.
